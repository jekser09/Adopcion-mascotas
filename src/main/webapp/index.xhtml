Adopcion de mascotas

Bienvenidos a la Aplicación para adopción de mascotas.

OBJETO: Se van a ingresar diferentes mascotas, cada mascota debe tener sus datos básicos, las personas se postulan para poder adoptar un tipo de mascota y en base a una puntuación que tendrá cada persona postulada se decidirá si es apta o no para adoptar a dicha mascota; en caso de que se genere la solicitud de adopción será aprobada o desaprobada y este evento deberá guardarse en un historial.
